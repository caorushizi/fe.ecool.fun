```js
const promise1 = new Promise((resolve, reject) => {
  console.log("promise1");
});
console.log("1", promise1);
```

## 过程分析：

- 从上至下，先遇到 new Promise，执行该构造函数中的代码 promise1
- 然后执行同步代码 1，此时 promise1 没有被 resolve 或者 reject，因此状态还是 pending

## 结果

```
'promise1'
'1' Promise{<pending>}
```
