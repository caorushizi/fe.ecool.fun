实现的方法很多，除了使用循环（for,while,forEach 等）外，最简单的是使用`Array.from`

```js
// 方法一
Array.from(new Array(10001).keys()).slice(1);

// 方法二
Array.from({ length: 10000 }, (node, i) => i + 1);
```
