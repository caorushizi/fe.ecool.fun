- 先来理解下 babel 到底是做什么的？

简单来讲，babel 解决语法层面的问题。用于将 ES6+的高级语法转为 ES5。

- babel polyfill 又是做什么的？

如果要解决 API 层面的问题，需要使用垫片。比如常见的有 babel-polyfill、babel-runtime 和 babel-plugin-transform-runtime。
