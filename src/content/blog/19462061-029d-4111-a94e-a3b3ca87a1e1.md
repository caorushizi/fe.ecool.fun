```javascript
// counter.js
let counter = 10;
export default counter;

// index.js
import myCounter from "./counter";
myCounter += 1;
console.log(myCounter);
```

引入的模块是 _只读_ 的: 你不能修改引入的模块。只有导出他们的模块才能修改其值。
当我们给`myCounter`增加一个值的时候会抛出一个异常： `myCounter`是只读的，不能被修改。
