- TypeScript Map 文件是一个源映射文件，其中包含有关我们原始文件的信息。
- .map 文件是源映射文件，可让工具在发出的 JavaScript 代码和创建它的 TypeScript 源文件之间进行映射。
- 许多调试器可以使用这些文件，因此我们可以调试 TypeScript 文件而不是 JavaScript 文件。
