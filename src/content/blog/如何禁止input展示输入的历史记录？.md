在输入 input 时会提示原来输入过的内容，还会出现下拉的历史记录，禁止这种情况只需在 input 中加入： autocomplete=“off”

```typescript
undefined;
```

autocomplete 属性是用来规定输入字段是否启用自动完成的功能。
