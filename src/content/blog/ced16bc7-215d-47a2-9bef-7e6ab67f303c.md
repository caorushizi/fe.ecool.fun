```js
async function fn() {
  // return await 1234
  // 等同于
  return 123;
}
fn().then(res => console.log(res));
```

## 解析

正常情况下，async 中的 await 命令是一个 Promise 对象，返回该对象的结果。

但如果不是 Promise 对象的话，就会直接返回对应的值，相当于 Promise.resolve()

## 结果

```
123
```
