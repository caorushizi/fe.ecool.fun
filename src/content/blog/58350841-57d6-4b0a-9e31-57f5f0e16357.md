```js
Promise.reject(1)
  .then(res => {
    console.log(res);
    return 2;
  })
  .catch(err => {
    console.log(err);
    return 3;
  })
  .then(res => {
    console.log(res);
  });
```

## 解析

因为 reject(1)，此时走的是 catch，且第二个 then 中的 res 得到的就是 catch 中的返回值。

## 结果

```
1
3
```
