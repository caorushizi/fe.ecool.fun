\== 中，左右两边都需要转换为数字然后进行比较。

\[\]转换为数字为 0。

!\[\] 首先是转换为布尔值，由于\[\]作为一个引用类型转换为布尔值为 true, 因此!\[\]为 false，进而在转换成数字，变为 0。 0 == 0 ， 结果为 true
