React 是一个简单的 javascript UI 库，用于构建高效、快速的用户界面。

它是一个轻量级库，因此很受欢迎。它遵循组件设计模式、声明式编程范式和函数式编程概念，以使前端应用程序更高效。

它使用虚拟 DOM 来有效地操作 DOM。

它遵循从高阶组件到低阶组件的单向数据流。
