- HTML：超文本标记语言，是语法较为松散的、不严格的 Web 语言；
- XML：可扩展的标记语言，主要用于存储数据和结构，可扩展；
- XHTML：可扩展的超文本标记语言，基于 XML，作用与 HTML 类似，但语法更严格。
