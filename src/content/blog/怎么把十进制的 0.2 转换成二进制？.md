进制转换是比较基础的，如果大家熟悉 js 的 API ，那么会首先想到这两个方法：

- 十进制转二进制：num.toString(2)
- 二进制转十进制：parseInt(num, 2)

所以答案就是 `(0.2).toString(2)`，可以简写为 `0.2.toString(2)`
