- 从 `构造函数` 获得 原型对象：

```
构造函数.prototype
```

- 从 `对象实例` 获得 `父级原型对象`：

```
方法一： 对象实例.__proto__        【 有兼容性问题，不建议使用】
方法二：Object.getPrototypeOf( 对象实例 )
```
