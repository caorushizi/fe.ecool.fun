类表示一组相关对象的共享行为和属性。

例如，我们的类可能是 Student，其所有对象都具有该 attendClass 方法。另一方面，John 是一个单独的 type 实例，Student 可能有额外的独特行为，比如 attendExtracurricular.

你使用关键字声明类 class：

```
class Student {
    studCode: number;
    studName: string;
    constructor(code: number, name: string) {
    	this.studName = name;
    	this.studCode = code;
    }
 }
```
