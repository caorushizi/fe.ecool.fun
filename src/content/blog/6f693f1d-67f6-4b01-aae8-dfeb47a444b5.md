因为浏览器在渲染 html 的时候是从上到下执行的，当遇到 js 文件的时候就会停止当前页面的渲染，转而去下载 js 文件。

如果将 script 标签放在头部，在文件很大的情况下将导致首屏加载时间延长，影响用户体验。

## 解决办法

- 将 script 标签放在 body 的底部
- 通过 defer、async 属性将 js 文件转为异步加载
