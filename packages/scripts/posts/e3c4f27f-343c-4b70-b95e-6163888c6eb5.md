```javascript
const set = new Set([1, 1, 2, 3, 4]);

console.log(set);
```

---

`Set`对象是独一无二的值的集合：也就是说同一个值在其中仅出现一次。
我们传入了数组`[1, 1, 2, 3, 4]`，他有一个重复值`1`.以为一个集合里不能有两个重复的值，其中一个就被移除了。所以结果是 `{1, 2, 3, 4}`.
