react 使用 diff 算法，使用 key 来做同级比对。如果使用数组下标作为 key，有以下情况：

- 在数组头部或中部插入或删除元素： 所有 key 对应的节点的值发生更改，进行重新渲染。造成性能损耗
- 而如果使用数组中唯一值来作为 key：不管是在何处插入或删除节点，其他 key 对应的节点的值未发生更改，只需插入或删除操作的数组节点。
