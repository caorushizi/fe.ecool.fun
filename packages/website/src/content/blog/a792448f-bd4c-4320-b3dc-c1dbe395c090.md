CSS 属性 `touch-action` 用于设置触摸屏用户如何操纵元素的区域(例如，浏览器内置的缩放功能)。

最简单方法是：

```css
html {
  touch-action: none;
  touch-action: pan-y;
}
```

还可以直接指定对应元素的宽度和 overflow：

```css
html {
  width: 100vw;
  overflow-x: hidden;
}
```
